Pertanyaan Umum

Temukan jawaban untuk pertanyaan yang sering diajukan

1. Sambungkan perangkat Anda ke jaringan WiFi yang tersedia
2. Buka browser dan halaman login akan muncul otomatis
3. Masukkan kode voucher yang telah Anda beli
4. Klik tombol "Login" dan Anda akan terhubung ke internet

Anda dapat membeli voucher melalui:
- Hubungi admin via WhatsApp
- Pembayaran via QRIS
- Transfer bank

Setelah pembayaran dikonfirmasi, kode voucher akan dikirimkan ke Anda.

Jika voucher tidak bisa digunakan, coba langkah berikut:
1. Pastikan kode voucher diketik dengan benar (perhatikan huruf besar/kecil)
2. Pastikan voucher belum digunakan sebelumnya
3. Pastikan voucher belum kadaluarsa
4. Coba refresh halaman login

Jika masih bermasalah, hubungi admin untuk bantuan.

Kecepatan internet bervariasi tergantung paket yang Anda pilih. Umumnya:
- Paket reguler: Up to 10 Mbps
- Paket premium: Up to 25 Mbps
- Paket unlimited: Up to 50 Mbps

Kecepatan aktual dapat bervariasi tergantung kondisi jaringan dan jumlah pengguna.

Satu voucher hanya dapat digunakan untuk satu perangkat dalam satu waktu. Jika Anda ingin menggunakan di beberapa perangkat, Anda perlu membeli voucher terpisah untuk setiap perangkat.

Untuk kebutuhan multi-device, silakan hubungi admin untuk paket khusus.

Untuk logout dari WiFi:
1. Buka halaman status dengan mengetik alamat IP router di browser
2. Klik tombol "Logout" di halaman status
3. Atau disconnect dari jaringan WiFi

Waktu voucher akan tetap berjalan jika Anda tidak melakukan logout.
